The test is run using the program "site_testing.py".
Enter the name of the test, the site's url you want to test and arguments (if needed).

If needed to run all tests use program "all_test.py". Enter site's url.

Python library's needed to install:
    -bs4
    -subprocess
    -selenium
    -requests_html
    -requests
    -pprint

Need to download webdriver for Chrome (chromedriver_linux64.zip): https://chromedriver.chromium.org/downloads
Place "chromedriver.exe" in root folder.
